#Rhythm-Slap
Brian Wang

Joe Wang

##Description
Using swipe gestures, slap someone to the beat and score combo points. 8-bit fun for all!

##Features
 - Rhythm Slapping
 - Four 8-beat combinations of slaps
 - Easy-to-add combination system.
 - Sound effects
 - Particle effects
 - BeatBorder
